# native-kakao-login

kakao login modules for react native

## Installation

```sh
npm install native-kakao-login
```

## Usage

```js
import NativeKakaoLogin from "native-kakao-login";

// ...

const result = await NativeKakaoLogin.multiply(3, 7);
```
